# Intelligent Quantum Materials Lab (IQM Lab) Homepage

A lightweight GitHub Pages starter site for the future Intelligent Quantum Materials Lab (IQM Lab). It is designed to be hosted first on GitHub Pages and later migrated to a Southeast University website.

## Quick start

1. Create a GitHub repository, e.g. `lin-group.github.io` or `lflin-lab-homepage`.
2. Upload all files in this folder.
3. In GitHub: **Settings → Pages → Build and deployment → Deploy from branch → main / root**.
4. Edit `_config.yml`, `index.html`, and files under `_data/` as the group grows.

## Structure

- `index.html` — homepage
- `assets/style.css` — visual design
- `assets/portrait.jpg` — PI portrait
- `_data/publications.yml` — featured publications
- `_data/openings.yml` — recruiting positions
- `_config.yml` — GitHub Pages metadata
